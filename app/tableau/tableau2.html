Movie Players

(shift-click to download the player)
tab00-07.zip Win32 (1700KB) Windows 95 or Windows NT
To install:
1. download the file
2. unzip it into a temp directory
3. run setup.exe
4. remove files from the temp directory

To run:
1. double-click Movie icon
2. type movie name from list above
3. sit back and watch

tab00-07.freebsd.tar.gz FreeBSD 2.1 (230KB)
To install:
1. download the file to some directory
2. cd to that directory
3. type: zcat < tab00-07.os-type.tar.gz | tar xf -

To run:
1. type: cd tab00-07
2. type: ./tableau -input movie (from list above)
3. sit back and watch
