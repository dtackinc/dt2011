For amusement purposes, we encoded some sequences using the encoder from Telenor.

You have to download an entire movie file first. Then run Telenor's decoder on it.
Get the original Norwegian copy or the local copy .

To run the decoder under Win32:
1. open the MS-DOS prompt under Windows 95 or Windows NT
2. cd to the directory where you downloaded the Telenor decoder zip file
3. pkunzip tmn_20.zip
4. tmndec -o6 -f2 erin12.263

We know there is room for improvement. Any suggestions are welcome!
